Theorem 1Euler’s Pentagonal Number Theorem

\begin{equation} \prod _{i=1}^{\infty } (1 - x^i) = 1+ \sum _{k=1}^{\infty } (-1)^k \left( x^{\frac{k(3k-1)}{2}} + x^{\frac{k(3k+1)}{2}} \right)=\sum _{k\in \mathbb {Z}} (-1)^k x^{\frac{3k^2-k}{2}}. \end{equation}
1

Proof ▶

We have

\begin{equation} \prod _{i=1}^{\infty } (1 - x^i) = \sum _{s=0}^{\infty } (-1)^s \sum _{1 \leq i_1 {\lt} i_2 {\lt} \ldots {\lt} i_s} x^{i_1 + i_2 + \ldots + i_s}. \end{equation}

Definition 2All Distinct Partitions

The set of partitions of \(n\) into distinct parts is defined as:

\begin{equation*} \text{all\_ distinct\_ partitions}(n) = \left\{ S \subseteq \mathbb {N} : \sum _{x \in S} x = n \text{ and } \forall x \in S, x {\gt} 0 \right\} \end{equation*}

Definition 3Even Distinct Partitions

The set of partitions of \(n\) into an even number of distinct parts is:

\begin{equation*} \text{even\_ distinct\_ partitions}(n) = \left\{ S \in \text{all\_ distinct\_ partitions}(n) : |S| \equiv 0 \pmod{2} \right\} \end{equation*}

Definition 4Odd Distinct Partitions

The set of partitions of \(n\) into an odd number of distinct parts is:

\begin{equation*} \text{odd\_ distinct\_ partitions}(n) = \left\{ S \in \text{all\_ distinct\_ partitions}(n) : |S| \equiv 1 \pmod{2} \right\} \end{equation*}

Definition 5Partition Counts

Let \(p_e(n)\) denote the number of partitions of \(n\) into an even number of distinct parts, and \(p_o(n)\) denote the number of partitions of \(n\) into an odd number of distinct parts:

\begin{align*} p_e(n) & = |\text{even\_ distinct\_ partitions}(n)| \\ p_o(n) & = |\text{odd\_ distinct\_ partitions}(n)| \end{align*}

Definition 6Pentagonal Numbers

The pentagonal numbers are given by:

\begin{equation*} \text{pentagonal}(k) = \left| \frac{3k^2 - k}{2} \right| \quad \text{for } k \in \mathbb {Z} \end{equation*}

Definition 7Pentagonal Number Coefficients

The pentagonal coefficient associated with \(k\) is:

\begin{equation*} \text{pentagonal\_ coeff}(k) = \left( (-1)^{|k|}, \text{pentagonal}(k) \right) \end{equation*}

Definition 8Pentagonal Series Coefficients

The coefficient of \(X^n\) in the pentagonal series is:

\begin{equation*} \text{pentagonal\_ series\_ coeff}(n) = \sum _{\substack {k \in \mathbb {Z} \\ \begin{bgroup} \text{pentagonal}(k) = n \end{bgroup}}} (-1)^{|k|} \end{equation*}

Theorem 9Euler’s Pentagonal Number Theorem (Distinct Partitions Version)

Let \(n\) be a natural number. Then:

\begin{equation*} p_e(n) - p_o(n) = \begin{cases} (-1)^k & \text{if } n = \frac{k(3k-1)}{2} \text{ for some } k \in \mathbb {Z} \\ 0 & \text{otherwise} \end{cases}\end{equation*}

Theorem 10Euler’s Pentagonal Number Theorem (Formal Power Series)

For a commutative ring \(R\), Euler’s pentagonal number theorem states:

\begin{equation*} \prod _{n=1}^{\infty } (1 - X^n) = \sum _{k \in \mathbb {Z}} (-1)^k X^{\text{pentagonal}(k)} \end{equation*}

where the product and sum are understood as formal power series in \(R\llbracket X \rrbracket \).

For a commutative ring \(R\) and natural number \(n\):

\begin{equation*} \left[ X^n \right] \prod _{k=1}^{\infty } (1 - X^k) = \text{pentagonal\_ series\_ coeff}(n) \end{equation*}
